# Banking_Management_System

I have created a small Banking Management System which has the following functionalities :- 
1) Deposit Money
2) Withdraw Money
3) Transfer Money
4) Account Details
5) Transaction Details
6) Exit


I have used basic C programming, Structures and File Handling in C.
